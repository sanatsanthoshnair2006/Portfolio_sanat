# Sanat Santhosh Nair - Portfolio Website

This is the repository for my portfolio website.
It was built strictly using HTML5 and CSS3.

## Objective

Create a well-structured portfolio website to showcase my skills, creativity, and previous projects, demonstrating expertise in HTML and CSS.

## Features & Implementation

1. **Website Structure**
   - Pure HTML & CSS implementation without external JS frameworks.
   - **Navigation Menu**: Functional links connecting sections natively (`#home`, `#education`, `#skills`, `#projects`, `#certifications`, `Resume/sanatresume.pdf`).
   - **Project Showcase**: A dedicated section displaying IoT, ANSYS, and Computer Vision projects.
   - **Downloadable Resume**: A fully functional `sanatresume.pdf` file located in the `Resume/` folder, linked directly from the navigation menu for easy downloading.

2. **Styling & Layout**
   - **CSS Grid** is actively utilized in the Skills section and Projects grid.
   - **Flexbox** is used for Navigation, Hero section, Timeline components, and Footer layout.
   - Modern techniques implemented:
     - **Hover Effects**: Added to links, buttons, and skill cards.
     - **Transitions & Animations**: Custom CSS animations (e.g., floating placeholder avatar).
     - **Gradient Backgrounds**: Used for text masking and dynamic accents.
     - **Card Layouts & Box Shadows**: Used for project elements and timeline cards.
   - **Responsive Design**: Custom Media Queries are implemented strictly ensuring correct scaling on devices smaller than `992px`, `768px`, and `576px`.

## Content

- **Home**: Personal details and objective.
- **Education**: B.Tech, HSC, and SSC timeline.
- **Skills**: Tools like Python, SQL, R, MATLAB, ANSYS, and Machine Learning.
- **Projects**: Highlights core academic and personal engineering achievements.
- **Certifications & Extra**: Achievements via DataCamp, Infosys Springboard, etc.
- **Resume Download**: Easily accessible resume PDF.
